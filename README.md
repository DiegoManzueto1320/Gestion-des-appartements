Ce projet a été développé dans le cadre académique, en vue de l’obtention du grade de Licencié en Informatique de Gestion, option Conception des Systèmes d’Information.

Il s’inscrit dans le domaine de la Direction de Gestion Immobilière et vise à fournir une solution logicielle pour la gestion des appartements.

🏠 Objectif du projet
Mettre en place un système permettant de gérer efficacement les appartements.

Offrir des fonctionnalités adaptées aux besoins de la gestion immobilière.

Servir de support pratique et académique pour l’apprentissage et la mise en œuvre des concepts de conception des systèmes d’information.

⚙️ Technologies utilisées
Langage de programmation : C#

Base de données : Mysql

Outils de développement : Archicteture MVC
